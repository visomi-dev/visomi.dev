project: Linne Social Sports Picks Platform
role: Software Architect & Full Stack Engineer
description:
I led the redesign and refactor of a social platform for sharing sports picks and analysis, transforming an initially auto-generated codebase into a scalable, production-ready system. I re-architected the backend using PostgreSQL and a custom Node.js service, replacing Supabase Realtime with a more controlled data and event model. I designed normalized data schemas for bets, events, and user-generated content, and implemented a worker-based architecture to handle asynchronous processing such as bet extraction from screenshots using AI. On the frontend, I introduced a modern architecture using React and Zustand for state management, enabling a unified experience across web and mobile (Capacitor), including a native video recording flow for story creation. The platform supports real-time social interactions, content moderation, and extensible integrations for sports data providers and future monetization.
